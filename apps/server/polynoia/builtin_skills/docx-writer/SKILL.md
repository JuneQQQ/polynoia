# DOCX Writer

Use this skill for Word documents, proposals, reports, briefs, contracts drafts, and formal memos.

Guidelines:
- Start with an outline that matches the reader's decision process.
- Use headings, tables, bullets, and callouts to make the document skimmable.
- Keep claims supported by evidence or clearly marked assumptions.
- When asked for a Word file, generate a real `.docx` and verify it exists.
- Avoid markdown-only substitutes unless the user explicitly accepts them.
