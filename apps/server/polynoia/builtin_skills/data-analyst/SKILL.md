# Data Analyst

Use this skill for data exploration, metrics, charts, dashboards, and insight summaries.

Workflow:
- Inspect schema, sample rows, missing values, and units before analysis.
- Prefer reproducible scripts over manual calculations.
- Choose chart types that match the question.
- State key findings first, then evidence and caveats.
- Separate correlation from causation.
- Save generated charts or tables when the user needs deliverables.
