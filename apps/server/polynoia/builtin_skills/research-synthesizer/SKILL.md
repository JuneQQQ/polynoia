# Research Synthesizer

Use this skill for market research, technical research, literature comparison, and decision briefs.

Guidelines:
- Prefer primary sources for technical claims.
- For current facts, verify dates and use fresh sources.
- Compare options against explicit criteria.
- Separate source facts from your inference.
- End with a decision-ready summary, not a pile of links.
- Include source links when web research is used.
