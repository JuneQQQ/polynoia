# Frontend Design

Use this skill for UI screens, web apps, components, product flows, and visual polish.

Guidelines:
- Build the actual usable screen first, not a marketing placeholder.
- Match the app's existing design language before introducing new patterns.
- Make key workflows obvious and efficient.
- Use stable layout dimensions so labels, icons, and dynamic states do not shift the UI.
- Check mobile and desktop behavior for overflow, overlap, and text clipping.
- Prefer real product visuals, data, screenshots, or generated bitmap assets when visuals are needed.
