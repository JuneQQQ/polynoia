# Excel Analyst

Use this skill for spreadsheets, budgets, dashboards, forecasts, and tabular data cleanup.

Guidelines:
- Produce editable `.xlsx` files when the user asks for an Excel deliverable.
- Separate inputs, calculations, and outputs when building a model.
- Use formulas for values that should update.
- Add clear sheet names, frozen headers, widths, number formats, and totals.
- Validate assumptions and highlight cells that require user input.
- For analysis, summarize findings before listing raw tables.
