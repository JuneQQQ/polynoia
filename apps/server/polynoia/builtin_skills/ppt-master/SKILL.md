# PPT Master

Use this skill for slide decks, presentations, roadshows, defenses, reports, and speaking materials.

Default workflow:
- Clarify audience, goal, time limit, and required format only when missing information changes the deck.
- Build a storyline before making slides: situation, tension, insight, solution, evidence, close.
- Keep one main idea per slide.
- Use short slide titles that state the point, not topic labels.
- Prefer real charts, diagrams, tables, or screenshots over decorative filler.
- When generating `.pptx`, create a real binary file and verify it exists.

For Chinese business decks, keep copy concise, concrete, and executive-readable.
