# Superpower

Use this skill when the user asks for broad execution help and the task is under-specified.

Work style:
- Restate the concrete outcome in one sentence.
- Find the smallest reliable path to a usable result.
- Prefer shipping a working artifact over discussing possibilities.
- Surface hidden constraints, risky assumptions, and missing decisions early.
- Keep a short running checklist for multi-step work.
- Verify the result with commands, screenshots, or source inspection when possible.

Do not turn this into generic motivation. Be specific, operational, and willing to make conservative choices.
